KanbanApp — это веб-приложение для управления задачами, которое позволяет пользователям создавать, обновлять и отслеживать свои задачи в удобном интерфейсе Kanban-доски. Приложение поддерживает авторизацию через cookies, обеспечивая безопасность данных. Пользователь может организовывать задачи по колонкам (например, "To Do", "In Progress", "Done") и изменять их состояние, перетаскивая между колонками. Цель приложения — упростить планирование и выполнение задач для повышения продуктивности.
